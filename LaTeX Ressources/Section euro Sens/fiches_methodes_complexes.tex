\sectionblack{Forme algébrique}
\defi{Forme algébrique}{L'écriture $x+\i y$ est appellée forme algébrique du complexe $z$.\\
$x$ s'appelle la \defin{partie réelle} et est noté $\Re(z)$.\\
$y$ s'appelle la \defin{partie imaginaire} et est notée $\Im(z)$. C'est un nombre réel.\\
$\bar{z}=a-b\i$ est le \defin{conjugué} de $z$. Il sert entre autre à mettre sous forme algébrique des fractions.}

\textbf{Calculs sur les formes algébriques : }\\
Les règles de calculs (addition, dévellopement, factorisation) valables pour les nombres réels sont aussi valables pour les nombres complexes.

\exosol{On donne $z_1=-2+3\i$ et $z_2=4-\i$.\\
Écrire sous forme algébrique : 
\begin{enumerate}  
\item $z_1+z_2$ ;
\item $2z_1$ ;
\item $z_1\times z_2$.
\end{enumerate}
}
{\begin{enumerate}
  \item $z_1+z_2=(-2+3\i)+(4-\i)=-2+4+\i(3-1)=2+2\i$ ;
  \item $2z_1=2(-2+3\i)=-4+6\i$ ;
  \item $z_1\times z_2=(-2+3\i)(4-\i)=-8+2\i+12\i-\i^2=-8+14\i-(-1)=-7+14\i$.
 \end{enumerate}
}

\textbf{Mise sous forme algébrique de fractions : }\\
On multiplie numérateur et dénominateur par le conjugué du dénominateur.

\exosol{Mettre sous forme algébrique les nombres $z_3=\dfrac{2}{3+\i}$ et $z_4=\dfrac{1+2\i}{5-6\i}$.}
{\\
$z_3=\dfrac{2}{3+\i}=\dfrac{2(3-\i)}{(3+\i)(3-\i)}=\dfrac{6-2\i}{9-\i^2}=\dfrac{6-2\i}{10}=\dfrac{3}{5}-\dfrac{1}{5}\i$.\\
$z_4=\dfrac{1+2\i}{5-6\i}=\dfrac{(1+2\i)(5+6\i)}{(5-6\i)(5+6\i)}=\dfrac{5+6\i+10\i+12\i^2}{5^2-(6\i)^2}=\dfrac{5+16\i-12}{25-36(-1)}=\dfrac{-7+16\i}{61}.$}

\sectionblack{Forme trigonométrique -- Forme exponentielle}
\defi{forme trigonométrique -- exponentielle}
{On appelle \defin{module} de $z$ le nombre réel $\sqrt{x^2+y^2}$. On le note \defin{$|z|$} ou \defin{$r$}.\\
Il s'agit de la longueur du segment OM.\\
On appelle \defin{argument} de $z$ une mesure de l'angle entre $\overrightarrow{Ox}$ et $\overrightarrow{OM}$.\\
On le note souvent \defin{$\theta$}.\\
L'écriture $[r\ ; \ \theta]=r(\cos\theta+\i\sin\theta)$ est appellée forme trigonométrique.\\
L'écriture $r\eit{}$ est appellée forme exponentielle.}


\textbf{Calculs sur les formes trigonométriques -- exponentielles : }\\
Lorsque l'on multiplie deux nombres complexes, on multiplie les modules et additionne les arguments.\\
Le conjugué de $z$ à même module mais un argument opposé à $z$.\\
Autrement dit : $r\eit{}\times r\eitp{}=r\times r'\e^{\i(\theta+\theta')}$ et $\bar{r\eit{}}=r\e^{-\i\theta}$.

\exosol{Soit $z=3\ei{3}{4}$ et $z'=7\ei{-2}{3}$.\\
Donner les formes exponentielles de  $zz'$ ; $\frac{z}{z'}$ ; $\frac{1}{z} ; \bar{z} ; z^8$}
{\\$zz'=3\times7e^{i(\cfrac{3\pi}{4}+\cfrac{-2\pi}{3})}=21\ei{}{12}$.\\[5pt]
$\cfrac{z}{z'}=\cfrac{3}{7}e^{i(\cfrac{3\pi}{4}-\cfrac{-2\pi}{3})}=\cfrac{3}{7}\ei{17}{12}$.\\[5pt]
$\cfrac{1}{z}=\cfrac{1}{3}\ei{-3}{4}$.\\[5pt]
$\bar{z}=\ei{-3}{4}$.\\[5pt]
$z^8=\pa{3\ei{3}{4}}^8=3^8\ei{3\times8}{4}=3^8\ei{6}{}=3^8$
}

\textbf{Mise sous forme algébrique de $z=[r;\theta]=r\eit{}$}
% {En considérant les projections orthogonales du point $M$ sur les axes $(O,\vect{u})$ et $(O,\vect{v})$, on obtient les relations :
% $$x=r\cos\theta \qquad y=r\sin\theta.$$
% On a alors $$z=x+\i y=r\cos\theta+\i r\sin\theta=r(\cos\theta+\i\sin\theta)$$}


\exosol{Donner la forme algébrique du nombre complexe $z_5$ de module $2$ et d'argument $\frac\pi3$.}
{\\\textbf{Méthode 1 : }\\
$z_5=r(\cos\theta +\i \sin\theta)=2\pa{\cos(\dfrac\pi3)+\i\sin(\dfrac\pi3)}=2\pa{\dfrac{1}{2}+\i\dfrac{\sqrt{3}}{2}}=1+\i\sqrt{3}$.\\
\textbf{Méthode 2 : }\\
$x=r\cos\theta=2\cos(\dfrac\pi3)=1$ et $y=r\sin\theta=2\sin(\dfrac\pi3)=\sqrt{3}$ donc $z=1+\i\sqrt{3}$. 
}

\textbf{Pour déterminer l'affixe d'un vecteur :} $z_{\vect{AB}}=z_B-z_A$.\\
\textbf{Pour trouver le milieu d'un segment  :} $I$ milieu de $[AB]$ d'où $Z_I=\dfrac{z_A+z_B}{2}$.
\textbf{Pour prouver des symétries : }$z$ et $\bar{z}$ sont symétriques par rapport à $(Ox)$\\[15pt]

\textbf{Pour prouver que des points sont sur un cercle : } On utilise les modules des affixes des points.

\textbf{Complexes et transformations : }\\
La translation de vecteur \vect{u} d'affixe $b$ a pour écriture complexe $z'=z+b$.\\
La rotation de centre O et d'angle $\theta$ a pour écriture complexe $z'=\eit{}\times z$.\\

\sectionblack{Complexes et équations}
\theo{Équations du second degré}{
On considère l'équation $az^2+bz+c=0$ où $a$, $b$ et $c$ sont trois nombres réels. \\
On pose $\Delta=b^2-4ac$. 
\begin{itemize}
 \itemb si $\Delta>0$, l'équation admet deux solutions $\cfrac{-b\pm\sqrt{\Delta}}{2a}$ ;
 \itemb si $\Delta=0$, l'équation admet une solution double $-\dfrac{b}{2a}$ ;
 \itemb si $\Delta<0$, l'équation n'admet pas de solution dans $\R$ mais admet deux solutions conjuguées dans $\C$ : $\cfrac{-b\pm\i\sqrt{-\Delta}}{2a}$.
\end{itemize}
}



\textbf{Identification :}

\exosol{Soit $P(z) =  z^3 - 4z^2 + 8z - 8$ où $z$ est une variable complexe.
\begin{enumerate} 
\item Calculer P(2). Que peut-on en déduire pour $P(z)$ ?
\item Trouver les nombres r\'eels $a,~ b$ et $c$ tels que, pour tout nombre complexe $z$,
\[P(z) = (z - 2)\left(az^2 + bz + c\right).\]
\item En déduire les solutions de l'\'equation $P(z) = 0$ dans l'ensemble $\C$ des nombres complexes.
\end{enumerate}}
{\begin{enumerate} 
\item %{Calculer $P(2)$. Que peut-on en déduire pour $P(z)$ ?}\\
$P(2)=0$ donc $P$ est factorisable par $z-2$.
\item %{Trouver les nombres r\'eels $a,~ b$ et $c$ tels que, pour tout nombre complexe $z$,
%\[P(z) = (z - 2)\left(az^2 + bz + c\right).\]}
$\begin{array}[t]{rcl}
P(z) &=&(z - 2)\left(az^2 + bz + c\right)\\
&=&z^3+bz^2+cz-2az^2-2bz-2c\\
&=&x^3+(b-2a)z^2+(c-2b)z-2c
\end{array}$\\
Par identification, on obtient :$\left\{\begin{array}{ccc}
				a &=& 1\\
				b-2a &=& -4\\
				c-2b &=& 8\\
				-2c &=& -8\\
				\end{array}\right.$ soit  $\left\{\begin{array}{ccc}
				a &=& 1\\
				b &=& -2\\
				c &=& 4
				\end{array}\right.$
$$P(z)=(z-2)(z^2-2z+4)$$
\item %{R\'esoudre alors l'\'equation $P(z) = 0$ dans l'ensemble $\C$ des	nombres complexes.}\\
$P(z)=(z-2)(z^2-2z+4)=0$.\\
Un produit de facteur est nul si et seulement si un des facteurs est nul.\\
\begin{tabular}{c|l}
$z-2=0$ & $ z^2-2z+4=0$\\
$z=2$ & $\Delta=b^2-4ac=(-2)^2-4\times1\times4=-12$\\
&$\Delta$ est négatif, l'équation a deux solutions complexes conjugués :\\
&$z_1=\cfrac{-b - \i\sqrt{-\Delta}}{2a}=\cfrac{2 - \i\sqrt{12}}{2}=1-i\sqrt{3}$ et $z_2=\bar{z_1}=1+\i\sqrt{3}$\\
\end{tabular}\\
l'équation a trois solutions : $z=2$ ou $z=1-\i\sqrt{3}$ ou $z=1+\i\sqrt{3}$.
\end{enumerate}}

\textbf{Remarque : } Afin d'obtenir la factorisation, une autre méthode consiste à effectuer la division des polynomes. Cette méthode est hors programme mais si vous la connaissez, elle est très efficace et peut s'utiliser dans d'autres situations.
