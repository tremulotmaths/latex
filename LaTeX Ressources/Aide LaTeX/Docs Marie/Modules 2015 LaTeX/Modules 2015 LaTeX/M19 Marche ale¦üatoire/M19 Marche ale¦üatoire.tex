\textcolor{blue}{\textbf{I. Présentation}}
\begin{enumerate}
\item Marche aléatoire\\
Une \emph{marche aléatoire} est un déplacement constitué de pas successifs, aléatoires et indépendants. Les applications des marches aléatoires sont multiples dans des domaines variés.\\[7pt]
\emph{Recherche à effectuer}
\begin{enumerate}
\item Faire une présentation du botaniste \textsc{Robert Brown}.
\item Le mouvement brownien
\begin{itemize}
\item Présenter la recherche de Robert Brown à l'origine de cette description
\item Préciser une utilisation de ce mouvement en physique, et une en mathématiques financières 
\end{itemize}
\end{enumerate}

\item Situation\\
Une puce se déplace sur un axe gradué; elle part de l'origine et, à chaque saut, elle se déplace de manière aléatoire de une unité, vers la gauche ou vers la droite.
\end{enumerate}
\vspace{20pt}

%_______________
\textcolor{blue}{\textbf{II. Simulation de quatre sauts}}\\[5pt]
Chaque suite de quatre sauts est appelée \emph{promenade de quatre sauts}.
\begin{enumerate}
\item Simuler un saut\\
Proposer une façon de simuler un saut, puis une instruction à saisir dans la calculatrice.
\item  Promenade de quatre sauts
	\begin{enumerate}
	\item Simuler cinq promenades de quatre sauts et indiquer la position d'arrivée pour chacune d'entre elles.
	\item La puce peut-elle atteindre toutes les positions? Justifier la réponse.
	\end{enumerate}
\item  Modélisation
	\begin{enumerate}
	\item A l'aide d'un arbre, obtenir toutes les promenades de quatre sauts et, pour chacune d'entre elles, indiquer la position d'arrivée correspondante.

\textcolor{blue}{\textbf{III. Simulation de dix sauts}}\\[5pt]
On considère maintenant des promenades de dix sauts.
\begin{enumerate}
\item Si on construisait un arbre pour représenter l'expérience et décrire ces promenades aléatoires, combien aurait-il de branches à son extrémité?\\
\emph{Observer les arbres représentant les promenades de deux sauts, de trois sauts, ... }
\item Compte tenu du très grand nombre de résultats possibles, on se propose d'utiliser un algorithme pour simuler ces promenades de dix sauts, puis estimer une loi de probabilité à partir des fréquences obtenues.\\[3pt]
\newpage
\textsc{Avec le logiciel Algobox}\\
Marche aléatoire de dix sauts
\begin{center}
   \begin{algobox}
\Variables
\Ligne x EST\_DU\_TYPE NOMBRE
\Ligne n EST\_DU\_TYPE NOMBRE
\Ligne s EST\_DU\_TYPE NOMBRE

\Ligne x PREND\_LA\_VALEUR 0
\Pour{n}{1}{10}
\DebutPour
\Ligne s PREND\_LA\_VALEUR ALGOBOX\_ALEA\_ENT(0,1)
\Si{(s==1)}
\DebutSi
\Ligne x PREND\_LA\_VALEUR x+1
\FinSi
\Sinon
\DebutSinon
\Ligne x PREND\_LA\_VALEUR x-1
\FinSinon
\FinPour
\Ligne AFFICHER "La puce se trouve au point d'abscisse: "
\Ligne AFFICHER x
\FinAlgo
\end{algobox}
\end{center}
