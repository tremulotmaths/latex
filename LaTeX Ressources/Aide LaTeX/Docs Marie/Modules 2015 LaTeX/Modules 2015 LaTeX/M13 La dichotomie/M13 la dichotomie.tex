\textcolor{blue}{\textbf{I. Situation}}\\[7pt]
L'objectif est de produire un algorithme permettant la recherche d'un nombre vérifiant certaines conditions: nombre à deviner, solution d'une équation \ldots
\vspace{15pt}

%_______________
\textcolor{blue}{\textbf{II. Le jeu du nombre à deviner}}\\[7pt]
On demande à un utilisateur de deviner un nombre choisi au hasard entre 10 et 100. Il n'a droit qu'à un certain nombre d'essais et, à chacune de ses propositions, on lui indique si le nombre cherché est plus grand, ou s'il est plus petit.
\begin{enumerate}
\item \begin{enumerate}
\item Faire plusieurs essais par groupe de deux, noter à chaque fois les nombres proposés et le nombre d'essais effectués pour parvenir à la solution.
\item Etablir une stratégie de recherche du nombre à deviner.
\end{enumerate}
\item Quel est le nombre minimum d'essais pour déterminer, à coup sûr, la solution? 
\item  On souhaite écrire un algorithme permettant de mettre en oeuvre ce jeu, et ne laissant à l'utilisateur qu'un nombre limité d'essais.
\begin{enumerate}
\item Réfléchir aux variables\\
 L'algorithme nécessite:
\begin{itemize}
\item une variable qui contiendra le nombre à deviner
\item une variable qui contiendra les différents essais de l'utilisateur
\end{itemize}
Par ailleurs, il faut définir:
\begin{itemize}
\item  un compteur qui augmente de 1 à chaque essai.
\end{itemize}
\item Réfléchir à la structure de l'algorithme\\
L'algorithme nécessite:
\begin{itemize}
\item une \emph{instruction conditionnelle} du type \textsc{ si ... alors ...}
\item une \emph{instruction alternative} du type \textsc{ si ... alors ... sinon ...}
\item une \emph{instruction itérative} du type \textsc{ tant que  ...}
\end{itemize}

%\item soit le nombre proposé est supérieur au nombre cherché, on affiche dans ce cas le message \og c'est plus petit!\fg.
%\item soit le nombre proposé est inférieur au nombre cherché, on affiche dans ce cas le message \og c'est plus grand!\fg.
%\end{itemize}
\end{enumerate}
\newpage

\item L'algorithme est traduit ci-dessous dans le langage du logiciel \textsc{Algobox}.
\begin{enumerate}
\item Cet algorithme se compose de plusieurs blocs, l'un présentant une instruction itérative, deux autres présentant des instructions conditionnelles.\\
 Les repérer et les délimiter à l'aide de couleurs.
\item Que représente la variable $S$? Expliquer l'instruction de la ligne 6 de l'algorithme.
\item Que représente la variable $n$? Quelle est sa valeur initiale?
\item Repérer la condition qui provoque l'arrêt du processus itératif. Quelle est la ligne qui indique cet arrêt?
\item Quelles sont les différentes sorties possibles pour cet algorithme?
\end{enumerate}
\end{enumerate}
\vspace{15pt}

\textcolor{blue}{\textbf{III. Recherche d'un zéro par dichotomie}}\\[7pt]
La dichotomie est utilisée lors de la recherche de solutions approchées d'une équation.\\[7pt]
Exemple\\
On considère la fonction $f$ définie sur $\R$ par $f(x)=-x^2+10x-23$.\\
On souhaite obtenir un encadrement d'amplitude $10^{-1}$ des solutions de l'équation $f(x)=0$.
\begin{enumerate}
\item Expliquer pourquoi l'intervalle $\Iff{3}{4}$ contient une solution de l'équation $f(x)=0$.
\item On coupe cet intervalle en deux. Comment déterminer si cette solution est contenue dans l'intervalle $\Iff{3}{3,5}$ ou dans l'intervalle $\Iff{3,5}{4}$?
\item Combien de fois faut-il répéter ce processus pour obtenir un intervalle d'amplitude $10^{-2}$?
\item Déterminer un autre intervalle d'amplitude 1 contenant une solution de l'équation $f(x)=0$, puis obtenir, comme précédemment, un intervalle d'amplitude $10^{-1}$.
\end{enumerate}
\vspace{7pt}
\textsc{Principe}\\%[2pt]
On dispose d'un intervalle $\Iff{a}{b}$ contenant une solution de l'équation $f(x)=0$. On partage cet intervalle en deux intervalles d'amplitude moitié $\Iff{a}{m}$ et $\Iff{m}{b}$ et on repère celui des deux qui contient la solution. On réitère ce processus jusqu'à obtenir la précision $10^{-p}$ souhaitée.\\[7pt]
L'algorithme est donné ci-dessous. Le compléter à chaque question.
\begin{enumerate}
\item Comment se calcule le nombre $m$ qui sépare l'intervalle $\Iff{a}{b}$ en deux intervalles de même amplitude? \\
Compléter la ligne 13.
\item Expliquer les instructions des lignes 14, 16 et 20.
\item  Quand arrête-t-on le processus itératif?\\
Compléter la ligne 11.
\end{enumerate}
\vspace{15pt}
