\textcolor{blue}{\textbf{I. Situation}}\\[10pt]
%
$ABCD$ est un carré.
\begin{enumerate}
\item Construire le point $I$, situé à l'intérieur du carré, tel que le triangle $DCI$ soit équilatéral.
\item Construire le point $J$, situé à l'extérieur du carré, tel que le triangle $BCJ$ soit équilatéral.
\item Que remarque-t-on?
\end{enumerate}
\vspace{20pt}

\textcolor{blue}{\textbf{II. Observer, conjecturer}}\\[5pt]
On utilise ici le logiciel \textsc{GeoGebra}.
\begin{enumerate}
\item Tracer un carré $ABCD$ de côté variable, noté $a$.
\item Construire les points $I$ et $J$ définis dans la situation ci-dessus.
\item Tracer la droite $(AI)$.
\item Faire varier le côté du carré et établir une conjecture au sujet des points $A$, $I$ et $J$.
\end{enumerate}

\textcolor{blue}{\textbf{III. Démontrer}}\\[5pt]
On se place dans le repère $(D\,; \,C\,,\,A)$.
\begin{enumerate}
\item Justifier que ce repère est orthonormé.
\item Déterminer les coordonnées des sommets du carré dans ce repère.
\item Calculer les coordonnées des points $I$ et $J$ dans ce repère.
\item Démontrer la conjecture faite précédemment.
\end{enumerate}
