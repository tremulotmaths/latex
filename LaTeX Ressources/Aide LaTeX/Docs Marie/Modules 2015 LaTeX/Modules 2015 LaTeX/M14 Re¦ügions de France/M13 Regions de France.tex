\textcolor{blue}{\textbf{I. Situation}}\\[7pt]
Sur le site de l'\textsc{INSEE}, on peut trouver des bases de données concernant l'évolution de la population des régions de France de 1999 à 2009.\\[3pt]
Sur internet, se connecter au site de l'\textsc{INSEE}:    \url{http://www.insee.fr/fr/themes} \\[3pt]
Choisir les entrées:
\begin{itemize}
\item Thème: Population
\item Type de produit: Chiffres clés
\item Niveau géographique: Comparaisons régionales
\end{itemize}
\vspace{5pt}
Choisir le document:
\begin{itemize}
\item Evolution de la population de 2007 à 2012: comparaisons régionales et départementales
\end{itemize}
\vspace{3pt}
